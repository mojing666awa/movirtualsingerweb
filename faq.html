安装与配置

MVS 支持哪些显卡？
NVIDIA: RTX 2060 (6GB) 及以上，CUDA 12.1+。 AMD: RX 6600 及以上，通过 ROCm 6.0 或 ZLUDA 加速。 Apple: M1/M2/M3/M4 全系列，通过 MPS 后端原生支持。
AMD 显卡如何配置 MVS？
Linux 用户推荐 ROCm：pip install torch --index-url https://download.pytorch.org/whl/rocm6.0。Windows 用户推荐使用 ZLUDA 包装层，无需修改代码即可获得接近原生的性能。详见教程 EP.01。
Apple Silicon 训练有什么注意事项？
MPS 后端目前对 FP16 支持不完全稳定，建议使用 precision: 32。M1 Pro/Max 及以上芯片体验更佳。统一内存架构下大模型训练不会爆显存，但速度约为同级 NVIDIA 的 60-70%。
Windows 上提示 CUDA 未找到怎么办？
请确认已安装 CUDA Toolkit 12.1，且 PyTorch 版本匹配：pip install torch --index-url https://download.pytorch.org/whl/cu121。MVS 的 MAS 模块已预编译，无需额外安装 Visual Studio。

数据与录制

用什么软件录制干声？
任意 DAW 均可：Audacity（免费）、Reaper、Adobe Audition、Logic Pro 等。导出为 44100Hz/16bit WAV 格式。MVS 不自带录音器，专注于合成引擎本身。
录音时需要注意什么？
使用电容麦克风 + 独立声卡，在安静环境中录制。避免后期降噪/去混响处理，高频损失会严重影响合成质量。每句音频前后保留 0.3s 静音段便于自动切分。
训练一个可用声库最少需要多少数据？
最低约 30 分钟干净干声可获得基本可用的声库。推荐 1-2 小时以获得细腻的情感表现力。MVS 的姿态空间预训练可在小数据集上显著提升音质。

模型导出与兼容

如何将 Bert-VITS2 声库用于 MVS？
请前往 Bert-VITS2 官方仓库，按照其 README 中的导出指引将 checkpoint 转为 ONNX 格式，再放入 MVS 的 checkpoints/ 目录即可加载。
VITS-Fast 模型也兼容吗？
兼容。请前往 VITS-Fast 官方仓库 获取导出脚本。注意 VITS-Fast 缺少姿态向量，导入后会初始化为默认值，建议在 MVS 中补充姿态微调。
不同显卡的训练速度对比？
以 1 小时数据集为例： • RTX 4090: ~2 小时 • RTX 2060: ~12-18 小时 • RX 7900 XTX (ROCm): ~8-10 小时 • M2 Max (MPS, FP32): ~15-20 小时

许可与商业化

MVS 引擎本身可以商用吗？
可以。MVS 引擎采用 Apache-2.0 / MIT 双许可，对生成内容无任何限制。但声库的许可由声库作者自行决定，使用前请务必查看该声库附带的 LICENSE 文件。
修改 MVS 源码后需要开源吗？
不需要。Apache-2.0 / MIT 均为宽松许可，允许闭源修改和商业分发。但我们欢迎回馈社区，提交 PR 让项目变得更好。
