❓ 常见问题

找不到答案？请前往 GitHub Issues 提问

安装与配置

MVS 最低需要什么显卡？
最低要求为 NVIDIA RTX 2060 (6GB VRAM) 或同等性能显卡。推荐使用 RTX 3060 12GB 及以上以获得流畅的训练体验。Apple Silicon M1/M2/M3 全系列原生支持。
RTX 2060 能正常训练吗？会不会爆显存？
可以正常训练。MVS 内置自适应批处理（Dynamic Batching）和 FP16 混合精度，RTX 2060 6GB 显存下可稳定运行。建议在 config 中设置 precision: 16 并适当减小 batch_size。
Windows 上提示 CUDA 未找到怎么办？
请确认已安装 CUDA Toolkit 12.1，且 PyTorch 版本匹配：pip install torch --index-url https://download.pytorch.org/whl/cu121。MVS 的 MAS 模块已预编译，无需额外安装 Visual Studio。
没有 NVIDIA 显卡可以用吗？
Apple Silicon Mac 可通过 MPS 后端训练。纯 CPU 仅建议用于推理调试，训练速度极慢（约为 RTX 2060 的 1/50）。AMD 显卡暂不支持，敬请期待 ROCm 适配。

训练与数据

训练一个可用声库最少需要多少数据？
最低约 30 分钟干净干声可获得基本可用的声库。推荐 1-2 小时以获得细腻的情感表现力。MVS 的姿态空间预训练可在小数据集上显著提升音质。
RTX 2060 训练一个声库大概要多久？
以 1 小时数据集、FP16 精度为例：姿态预训练约 8-12 小时，端到端微调约 4-6 小时。总计约 12-18 小时。建议使用 nohup 或 tmux 后台运行。
支持哪些语言的声库训练？
目前官方支持中文（普通话）、日语、英语。其他语种可通过自定义音素表与 G2P 模块扩展，详见教程 EP.05。
数据集中有底噪/混响怎么办？
MVS 预处理脚本内置降噪模块：mvs-preprocess --denoise --dereverb。但强烈建议使用干净录音棚素材，后期降噪会损失高频细节。

推理与编辑器

MVS 必须搭配 OpenUtau 使用吗？
不必须。OpenUtau 是推荐的图形化编辑器，但 MVS 也提供 Python SDK 和命令行推理工具，可独立使用。ONNX 模型也可集成到其他 DAW 中。
推理时延迟很高怎么办？
RTX 2060 推理单句约 200-400ms，属于正常范围。若延迟过高，请检查是否误用 CPU 模式，或在 OpenUtau 设置中切换至 GPU 渲染后端。
生成的歌声有电音/机械感？
通常是训练步数不足或数据质量问题。建议检查 TensorBoard 中的 mel loss 曲线是否收敛，并确认训练数据采样率为 44100Hz。也可尝试增大姿态空间的维度。

许可与商业化

MVS 引擎本身可以商用吗？
可以。MVS 引擎采用 Apache-2.0 / MIT 双许可，对生成内容无任何限制。但声库的许可由声库作者自行决定，使用前请务必查看该声库附带的 LICENSE 文件。
我训练的声库可以出售吗？
完全可以。只要您拥有训练数据的合法授权，声库的版权归您所有。我们鼓励在社区中分享和销售优质声库。
修改 MVS 源码后需要开源吗？
不需要。Apache-2.0 / MIT 均为宽松许可，允许闭源修改和商业分发。但我们欢迎回馈社区，提交 PR 让项目变得更好。
