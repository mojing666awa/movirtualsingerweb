安装与配置

MVS 支持哪些显卡？
NVIDIA: GTX 2060 8GB 及以上（推荐 RTX 3060+）。Apple Silicon: M1/M2/M3 全系列原生支持。CPU 仅建议用于调试。
Windows 上提示 CUDA 未找到怎么办？
请确认已安装 CUDA Toolkit 12.1，且 PyTorch 版本匹配：pip install torch --index-url https://download.pytorch.org/whl/cu121

许可与商业化

MVS 生成的歌曲可以商用吗？
引擎本身对生成内容无限制。但声库的许可由声库作者自行决定，使用前请务必查看该声库附带的 LICENSE 文件。
